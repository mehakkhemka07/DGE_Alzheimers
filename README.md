Differential Gene Expression Analysis in Alzheimer’s Disease
This project identifies differentially expressed genes in Alzheimer’s disease using the GSE5281 from the Gene Expression Omnibus. Data is preprocessed and analyzed in R to detect significant gene expression changes, visualized using a volcano plot.

⚙️ Workflow
Data retrieval from GEO
Preprocessing & normalization
Sample grouping (AD vs Control)
Differential expression analysis in R
Statistical filtering (adjusted p-value, logFC)
Visualization using volcano plot

🧬 Dataset
161 brain tissue samples
74 Alzheimer’s vs 87 control
Platform: Affymetrix U133 Plus 2.0

🛠️ Tools & Technologies
R
GEOquery
limma (for differential analysis)
EnhancedVolcano

📌 Key Features
Uses real-world public transcriptomic data
Implements standard bioinformatics pipeline
Reproducible analysis workflow in R
